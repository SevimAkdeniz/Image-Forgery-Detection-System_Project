Çalıştırma:
1) python -m venv venv
2) Windows: venv\Scripts\activate
   Mac/Linux: source venv/bin/activate
3) pip install -r requirements.txt
4) python run.py
5) Tarayıcı: http://127.0.0.1:5000

Not:
- Tek görsel yüklenirse copy-move/sahtecilik analizi yapılır.
- Orijinal + şüpheli görsel yüklenirse iki görüntü karşılaştırılır.
- Desteklenen formatlar: jpg, jpeg, png, bmp, webp, gif, tif, tiff.
